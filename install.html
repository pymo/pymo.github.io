pymo安装方法

Android版

【安装方法】
1. 安装apk。如果有旧版本最好先卸载再安装新版本。
2. 下载游戏数据包，在电脑上解压，然后拷贝到手机里。注意路径里不能有中文。
3. 在手机上打开pymo，选择游戏所在的文件夹就会开始游戏。


【按键说明】
快进：快速跳过对话
自动：自动播放/停止自动播放
上下左右确定键：上下左右确定
左方向键：隐藏/恢复对话框
返回键：返回/退出
菜单键：系统菜单
音量键：降低/升高音量

Symbian版

【安装方法】
（已安装pymo 旧版的可以跳过第1步）
1.依次在手机上安装游戏引擎压缩包里的pips.sis、Python_2.0.0.sis（如果装pips.sis显示更新错误，可以不安装pips.sis），推荐安装到C盘

2.安装pymo_v1_0_0.sis，要和Python平台安装在同一盘符下。之前安装过旧版本的pymo的话，覆盖安装即可。

3.在电脑上解压游戏数据压缩包，把解压出来的Python文件夹放到存储卡（E盘）的根目录下。如果提示是否覆盖，覆盖无妨。（注意：不要用手机直接解压这么大的文件，很有可能解压失败。）

4.运行pymo即可进行游戏

【按键说明】
导航键、2/8/4/6/5键：上下左右确定键
1键：快速跳过对话
3键：自动播放/停止自动播放
0键：系统菜单
*键：降低音量
#键：升高音量
左方向键：隐藏/恢复对话框

【安装常见问题解答】
一、安装了以后程序打不开、点击pymo没反应？
应该是Python平台没有安装好，可以在装好python平台之后，安装压缩包内的python前端界面（PythonScriptShell_2.0.0_3_0.sis）试一试，看看这个前端界面能不能运行，如果不能的话就是python没有安装好。解决方案：
1、之前装过低版本的Python平台的一定要完全删除其本体及所有组件之后再安装新版（包括所谓的Super Python 2.5版什么的，那些都不是官方版本，官方只出到2.0版）
2、python平台和pymo一定要装在同一盘符下
3、如果都装在C盘下不行，可以尝试都装在E盘

二、打开pymo提示No game found或者Can not find icon_mask.png?
说明Python平台装好了，但是没有把游戏资源放在存储卡正确的路径上。正确的方法应该是把Python文件夹放到存储卡的根目录下。看上去应该是这样 E://Python//游戏目录...

三、运行pymo后，可以玩了，但运行一段时间之后跳出?
1、可能是游戏资源没有解压好，不要用手机解压资源。
2、可能是运行内存不足，请确保空闲的运存至少有12M，可以尝试关闭游戏设置中的“资源预取功能”看情况是否有改善。
